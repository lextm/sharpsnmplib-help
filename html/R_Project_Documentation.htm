This is an SNMP library for both .NET and Xamarin/Mono.
Namespaces
NamespaceDescription
Lextm.SharpSnmpLib
Basic building blocks for SNMP packet parsing. ISnmpData derived classes such as Counter32 are matching SNMP data types. ISnmpPdu derived classes are matching SNMP PDU types.
Lextm.SharpSnmpLib.Messaging
The classes and interfaces under this namespace are used for SNMP message processing. ISnmpMessage derived classes are matching SNMP message types.
Lextm.SharpSnmpLib.Objects
Samples objects for #SNMP agent.
Lextm.SharpSnmpLib.Pipeline
All classes and interfaces under this namespace are used for SNMP engine implementation. The SNMP processing pipeline is similar to ASP.NET HTTP message processing pipeline.
Lextm.SharpSnmpLib.Security
All classes and interfaces under this namespace are used for SNMP v3 message handling.
